# Travelling Salesman Problem

A Python implementation of several algorithms for solving the Travelling Salesman Problem (TSP). This project compares heuristic and exact approaches for finding efficient routes between cities.

## Features

- Nearest Neighbor heuristic
- Prim-based approximation
- 2-opt optimization
- Branch and Bound exact solver
- Performance comparison between algorithms

## Technologies

- Python

## Installation

Clone the repository:

```bash
git clone https://github.com/kenzaayousfi/travelling-salesman-problem.git
cd travelling-salesman-problem
```

Run the project:

```bash
python3 src/main.py
```

## Algorithms

This project implements:

- Nearest Neighbor (PPP)
- Prim-based approximation
- 2-opt optimization
- Branch and Bound (HDS)
